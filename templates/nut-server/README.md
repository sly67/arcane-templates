# NUT Server

The purpose of NUT Server is to monitor a UPS attached device

![Logo](https://raw.githubusercontent.com/pi-hosted/pi-hosted/master/images/nut-server.png)

## Docker Image

`instantlinux/nut-upsd:latest`

## Categories

- Other
- Tools

## Ports

- 3493:3493/tcp

## Environment Variables

- `PUID` - PUID (default: `1000`)
- `PGID` - PGID (default: `1000`)
- `SERIAL` - SERIAL (default: `none`)
- `NAME` - NAME (default: `none`)
- `VENDOR ID` - VENDOR ID (default: `none`)
- `API_PASSWORD` - API_PASSWORD (default: `none`)
- `DESCRIPTION` - DESCRIPTION (default: `none`)
