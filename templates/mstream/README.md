# Mstream

Mstream is a personal music streaming server. You can use mStream to stream your music from your home computer to any device, anywhere. There are mobile apps available for both Android and iPhone.

![Logo](https://raw.githubusercontent.com/Qballjos/portainer_templates/master/Images/mstream.png)

## Docker Image

`linuxserver/mstream:latest`

## Categories

- Musicserver

## Ports

- 3000:3000/tcp

## Volumes

- `/portainer/Files/AppData/Config/Mstream` → `/config`
- `/portainer/Downloads` → `/music`
