# Komga

Koomga Manga, Comic and E-Book Server

![Logo](https://raw.githubusercontent.com/gotson/komga/master/.github/readme-images/app-icon.png)

## Docker Image

`gotson/komga:latest`

## Categories

- Libraryserver

## Ports

- 8080:8080/tcp

## Environment Variables

- `PUID` - PUID (default: `1000`)
- `PGID` - PGID (default: `100`)

## Volumes

- `undefined` → `/books`
- `undefined` → `/comics`
- `undefined` → `/manga`
- `undefined` → `/config`
