# Deluge

Deluge is a lightweight, Free Software, cross-platform BitTorrent client providing: Full Encryption, WebUI, Plugin System, Much more...

![Logo](https://raw.githubusercontent.com/Qballjos/portainer_templates/master/Images/deluge-icon.png)

## Docker Image

`linuxserver/deluge:latest`

## Categories

- Torrentserver

## Environment Variables

- `PUID` - PUID (default: `1000`)
- `PGID` - PGID (default: `100`)
- `UMASK_SET` - UMASK_SET

## Volumes

- `/portainer/Files/AppData/Config/Deluge` → `/config`
- `/portainer/Downloads` → `/downloads`
