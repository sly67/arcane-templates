# Rclone Config Backup

Rclone is a command line program to sync files and directories to and from cloud providers (Dropbox, GDrive, Box, etc)

![Logo](https://mediadepot.github.io/templates/img/rclone.png)

## Docker Image

`mediadepot/rclone:latest`

## Categories

- Backup
- Cloud
- Networkother
- Tools

## Volumes

- `/opt/mediadepot/apps/rclone` → `/srv/rclone/config`
- `/opt/mediadepot/apps` → `/mnt/data`
