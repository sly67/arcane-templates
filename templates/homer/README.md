# Homer

A dead simple static HOMepage for your servER to keep your s ervices on hand, from a simple yaml configuration file.

![Logo](https://raw.githubusercontent.com/Qballjos/portainer_templates/master/Images/homer.png)

## Docker Image

`b4bz/homer:latest`

## Categories

- Dashboardserver

## Ports

- 8902:8080/tcp

## Volumes

- `/portainer/Files/AppData/Config/Homer/assets` → `/www/assets`
- `/portainer/Files/AppData/Config/Homer` → `/www/config.yml`
