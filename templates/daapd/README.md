# Daapd

DAAP (iTunes) media server with support for AirPlay devices, Apple Remote (and compatibles), MPD and internet radio.

![Logo](https://raw.githubusercontent.com/linuxserver/beta-templates/master/lsiodev/img/daapd-icon.png)

## Docker Image

`linuxserver/daapd:latest`

## Categories

- Musicserver

## Environment Variables

- `PUID` - PUID (default: `1000`)
- `PGID` - PGID (default: `100`)

## Volumes

- `/portainer/Files/AppData/Config/Daapd` → `/config`
- `/portainer/Music` → `/music`
