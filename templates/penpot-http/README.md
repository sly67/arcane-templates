## Environment Variables

- `PUID` - PUID (default: `1000`)
- `PGID` - PGID (default: `1000`)
- `PENPOT_FLAGS_FE` - Frontend Flags (default: `enable-registration disable-secure-session-cookies enable-insecure-register enable-login-with-password`)
- `PENPOT_FLAGS_BE` - Backend Flags (default: `enable-registration disable-secure-session-cookies enable-insecure-register disable-email-verification`)
- `PUBLIC_URI` - Public URI (default: `http://192.168.1.1:8624`)
- `TELEMETRY_ENABLED` - Penpot Telemetry Enabled?
- `PORT` - Http Port (default: `8624`)
