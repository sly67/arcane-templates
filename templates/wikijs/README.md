# Wikijs

Wikijs A modern, lightweight and powerful wiki app built on NodeJS.

![Logo](https://raw.githubusercontent.com/Qballjos/portainer_templates/master/Images/wikijs.png)

## Docker Image

`linuxserver/wikijs:latest`

## Categories

- Notesserver

## Ports

- 3100:3000/tcp

## Volumes

- `/portainer/Files/AppData/Config/Wikijs` → `/config`
- `/portainer/Files/AppData/Config/Wikijs/data` → `/data`
