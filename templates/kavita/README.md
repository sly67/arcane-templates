# Kavita

Kavita Manga, Comic and E-Book Server

![Logo](https://raw.githubusercontent.com/Kareadita/Kavita/main/UI/Web/src/assets/images/logo.png)

## Docker Image

`kizaing/kavita:latest`

## Categories

- Libraryserver

## Ports

- 5000:5000/tcp

## Environment Variables

- `PUID` - PUID (default: `1000`)
- `PGID` - PGID (default: `100`)

## Volumes

- `undefined` → `/books`
- `undefined` → `/comics`
- `undefined` → `/manga`
- `undefined` → `/config`
