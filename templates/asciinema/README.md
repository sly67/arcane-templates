## Environment Variables

- `PUID` - PUID (default: `1000`)
- `PGID` - PGID (default: `1000`)
- `HOST_NAME` - Hostname (default: `localhost`)
- `HOST_PORT` - Host Port (default: `3033`)
- `SECRET_KEY` - Secret Application Key (default: `please-change-this-to-a-random-secret`)
