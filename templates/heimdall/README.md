# Heimdall

Heimdall is a way to organise all those links to your most used web sites and web applications in a simple way.

![Logo](https://mediadepot.github.io/templates/img/heimdall-icon.png)

## Docker Image

`linuxserver/heimdall:latest`

## Categories

- Tools
- Networkweb
- Networkother
- Statusstable

## Ports

- 80/tcp
- 443/tcp

## Environment Variables

- `PGID` - PGID (default: `15000`)
- `PUID` - PUID (default: `15000`)
- `TZ` - TZ (default: `America/Los_Angeles`)
