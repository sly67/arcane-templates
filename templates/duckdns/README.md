# DuckDNS

Duckdns is a free service which will point a DNS (sub domains of duckdns.org) to an IP of your choice.

![Logo](https://mediadepot.github.io/templates/img/duckdns.png)

## Docker Image

`linuxserver/duckdns:latest`

## Categories

- Networkdns
- Tools

## Environment Variables

- `SUBDOMAINS` - SUBDOMAINS
- `TOKEN` - TOKEN
- `TZ` - TZ (default: `America/Los_Angeles`)
