# YouTubeDL-Material

YoutubeDL-Material is a Material Design frontend for youtube-dl. It's coded using Angular 9 for the frontend, and Node.js on the backend.

![Logo](https://raw.githubusercontent.com/Qballjos/portainer_templates/master/Images/ytdlm.png)

## Docker Image

`tzahi12345/youtubedl-material:latest`

## Categories

- Downloaders

## Ports

- 17442:17442/tcp

## Volumes

- `/portainer/Files/AppData/Config/YTDLM` → `/app/appdata`
- `/portainer/Files/AppData/Youtube/Video` → `/app/video`
- `/portainer/Files/AppData/Youtube/Subscriptions` → `/app/subscriptions`
- `/portainer/Files/AppData/Youtube/Users` → `/app/users`
- `/portainer/Files/AppData/Youtube/Audio` → `/app/audio`
