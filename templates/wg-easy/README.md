## Environment Variables

- `WG_HOST` - WG_HOST (default: `example.domain.com`)
- `PASSWORD` - PASSWORD (default: `ENTER AN ADMIN PASSWORD`)
- `WG_PORT` - WG_PORT (default: `51820`)
- `WG_DEFAULT_DNS` - WG_DEFAULT_DNS (default: `1.1.1.1`)
- `WG_DEFAULT_ADDRESS` - WG_DEFAULT_ADDRESS (default: `10.8.0.x`)
- `WG_ALLOWED_IPS` - WG_ALLOWED_IPS (default: `0.0.0.0/0, ::/0`)
