# Seafile

Seafile is an open-source, cross-platform file-hosting software system. Files are stored on a central server and can be synchronized with personal computers and mobile devices through apps. Files on the Seafile server can also be accessed directly via the server's web interface.

![Logo](https://raw.githubusercontent.com/SelfhostedPro/selfhosted_templates/master/Images/seafile.png)

## Environment Variables

- `TIME_ZONE` - Time Zone (default: `Etc/UTC`)
- `MYSQL_ROOT_PASSWORD` - Database Password (default: `password`)
- `ADMIN_EMAIL` - Administrator Email (default: `me@example.com`)
- `ADMIN_PASSWORD` - Administrator Password (default: `asecret`)
- `SERVER_HOSTNAME` - Domain Name (default: `docs.seafile.com`)
- `ENABLE_HTTPS` - Enable HTTPS?
